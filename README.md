# GitHuberMessageFind  
> 一行命令输入 GitHub 用户名，即刻查看 Ta 的「公开资料 + 仓库统计 + 社交账号」——纯前端，零配置，打开即玩。

---

## ✨ 特性
- 🔍 **实时搜索** - 输入用户名即刻查询  
- 🌙 **毛玻璃 + 渐变动效** - 颜值在线  
- ⚡ **防抖 + 加载动画** - 不卡不闪  
- 🌍 **国际化日期** - 中文年月日友好显示  

---

## 🚀 5 秒上手
1. 克隆仓库  
   ```bash
   git clone https://github.com/WindySaber/GithuberMessageFind.git
   ```
2. 进入目录双击 `index.html` **或用任意静态服务器**  
   ```bash
   npx serve          # npm i -g serve 后可省略安装
   # 打开 http://localhost:3000 即可
   ```
3. 在输入框里敲 GitHub 用户名 → 回车 → 完成！

---

## 🛠 技术栈
- 原生 **HTML + CSS + JavaScript**（无框架，无构建）  
- **CSS 变量** 实现主题
- **Fetch + Async/Await** 调用 GitHub REST API v3
